# AI Resume Builder Pro

A modern, responsive web application that helps users create professional resumes with live preview, multiple templates, dark mode, PDF download, and automatic data saving.

---

## 📌 Features

- 🎨 Modern and responsive user interface
- 👤 Profile photo upload
- 📝 Live resume preview
- 🎯 Multiple resume templates
- 🌙 Dark / Light mode
- 💾 Auto Save using Local Storage
- 📄 Download resume as PDF
- 💼 Dynamic Projects section
- 🤖 Resume analysis and suggestions
- ✨ Smooth animations and toast notifications

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- html2pdf.js

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start creating your resume.
